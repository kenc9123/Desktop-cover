# 2026-06-29 开发日志

## 背景

用户希望寻找一种开源的 Windows 11 个性化桌面方案，但目标不是 Rainmeter 这类小工具、皮肤或壁纸，而是更接近系统原生的桌面交互层：

- 简约、克制、可互动。
- 支持鼠标驱动的视差和层级运动。
- 视觉气质接近《明日方舟：终末地》式工业科幻 UI。
- 能隐藏或弱化原生任务栏，提供 Dock、启动器、状态信息等系统级入口。
- 不希望依赖传统桌面挂件堆叠。

用户提到 `sonic-topography` 作为视觉参考。公开搜索暂未定位到明确的开源仓库，后续如用户提供链接，需要继续补充源码级调研。

## 今日调研结论

- Rainmeter、Lively Wallpaper、JaxCore 等更偏挂件或壁纸层，暂不作为核心路线。
- Seelen UI 是目前最接近桌面环境增强器的开源项目，具备 Dock、任务栏替换、启动器、窗口管理和小组件能力，但视觉方向不一定符合目标。
- Cairo Desktop 更像传统 Windows Shell 替换，生产力导向明显，不符合目标的未来感交互层。
- GlazeWM、komorebi、Whim 等更偏窗口管理，不承担完整视觉系统。
- Windhawk、ExplorerPatcher 等能修改 Explorer/任务栏行为，但不适合作为统一桌面体验的主底座。
- 根据用户提供的抖音链接，解析到作品 `7656117552200846690`，项目确认为 `yin-yizhen/sonic-topography`。
- `sonic-topography` 是 Electron + React + Three.js 的桌面音乐播放器和 3D 音频可视化应用，适合做视觉参考，不适合作为 TopoShell 的系统桌面底座。
- 用户进一步明确审美方向为“工业机能风，简约不失复杂”。

## 初步方向

创建 `TopoShell` 作为规划项目，定位为 Windows 11 的轻量桌面交互层，而不是完整替换 `explorer.exe` 的 Shell Replacement。

第一阶段只做规划和技术验证，不修改系统设置，不 hook Explorer，不写注册表。
