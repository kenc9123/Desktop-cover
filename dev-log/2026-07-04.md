## Desktop Replacement Feedback

The user tested the Explorer replacement direction and identified two shell-mode problems:

- After Explorer is closed, TopoShell still opens like a centered window instead of covering the whole desktop.
- The title-bar close button can leave the session without a desktop if it closes TopoShell while Explorer is not running.
- The minimize button is not useful for a desktop replacement surface.

Adjustment plan:

- Start TopoShell as a full-screen primary-display desktop surface.
- Remove the title-bar minimize button.
- Change shell exit paths so they start `explorer.exe` first when Explorer is not running.
- Keep the command/tray controls, but make close/exit behavior safer in shell mode.
