## 桌面替代反馈

用户测试 Explorer 替代方向后指出两个 Shell 模式问题：

- 关闭 Explorer 后，TopoShell 仍然像居中窗口一样打开，没有覆盖整个桌面。
- 右上角关闭按钮如果在没有 Explorer 的情况下关闭 TopoShell，会让当前会话没有桌面。
- 最小化按钮对桌面替代层没有意义。

调整计划：

- 启动后让 TopoShell 作为主屏全屏桌面层显示。
- 移除标题栏最小化按钮。
- 修改关闭/退出路径：如果 Explorer 没有运行，先启动 `explorer.exe`，再关闭 TopoShell。
- 保留命令面板和托盘控制，但让关闭/退出行为在 Shell 模式下更安全。

## 桌面替代修正已完成

- 将主窗口从居中窗口启动改为手动设置主屏全屏尺寸。
- 将窗口 resize mode 改为 `NoResize`。
- 移除标题栏最小化按钮。
- 移除命令面板里的隐藏到托盘动作。
- 移除托盘菜单里的隐藏/显示项，因为在 Shell 模式下隐藏本体可能留下空会话。
- 禁用标题栏拖动，避免全屏桌面层被拖离屏幕。
- 右上角关闭按钮和命令面板退出按钮现在都会走安全 Shell 退出路径。
- 增加 Explorer 兜底行为：
  - 如果没有 `explorer.exe` 进程，TopoShell 会先启动 `explorer.exe` 再关闭。
  - `Closing` 事件也会为非按钮关闭路径尝试执行兜底。
- 更新了中英文 README 和 Shell 替换说明文档。
