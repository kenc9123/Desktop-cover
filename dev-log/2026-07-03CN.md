# 2026-07-03 开发日志

## 目标

根据用户提供的《明日方舟：终末地》参考视频，重新调整鼠标视差动态效果。

用户明确指出当前动态效果不符合预期，希望直接参考视频里的鼠标视差感觉。

## 范围

- 所有项目改动继续限制在 `F:\Workspace\Personal\TopoShell` 内。
- 除非运动逻辑需要，不改动当前圆盘 UI 结构。
- 本轮重点放在动态行为：
  - 更平滑的目标跟随
  - 带一点延迟和惯性的响应
  - 不同层级有不同深度的视差
  - 轻微 3D 姿态变化
  - 不再把鼠标位置直接、僵硬地映射到界面 transform

## 实现计划

- 将即时鼠标映射改为 target/current 两组视差状态。
- 每一帧让 current 视差缓动追向 target。
- 加入小幅速度项，让运动有重量和回弹感。
- 背景网格、地形线、侧边栏和核心舞台使用不同视差深度。
- 保持圆盘信息可读，同时增加接近参考视频的层次运动。

## 已完成

- 增加 `MouseLeave` 处理，鼠标离开窗口后视差目标会回到中心。
- 将 `MouseMove` 中直接写 transform 的逻辑替换为：
  - `_parallaxTarget`
  - 平滑后的 `_parallax`
  - `_parallaxVelocity`
- 在渲染循环中加入弹簧阻尼式运动。
- 增加 `SmoothStep` 曲线，让鼠标运动有缓入缓出，而不是线性硬映射。
- 拆分层级深度：
  - 细网格作为远背景层移动
  - 地形线场作为近背景层移动
  - 左右侧栏作为前景 UI 板移动
  - 中央圆盘舞台作为主焦点层移动
- 将速度项加入圆盘的俯仰、偏航和滚转，让它有轻微拖拽重量，同时不牺牲可读性。
